Self-contained repo for building the weight stationary systolic array
VLSI capstone project for EE 478/526 B

Steps to get it working:
* Clone the repo and sub-modules: 
`git clone --recursive https://github.com/YaraAlShorman/systolic-array-ws.git`
* Go into ee477-hammer-cad and build it
`cd ee477-hammer-cad; make`
* Start building!
